# COPY AND MESSAGING FRAMEWORK (AGENT 5 - TRANSLATION LAYER)

## Objective
Translate ICP thinking into language that feels instantly recognizable.

## Inputs required
- Offer framework output
- Strategy output
- Research output

## Core rules
- Simple over smart
- Clear over persuasive
- Their language over your language
- Specific over broad
- One idea per message
- Worldview first

## Steps
1. Simulate internal monologue (5-10 raw thoughts)
2. Reuse real language, do not over-polish
3. Match message to awareness level
4. Build message using thought, reality, solution, low-friction CTA
5. Insert offer so it feels like their idea
6. Keep claims believable
7. Use context only if verified
8. Use subtle tension
9. Generate 5-10 distinct variants
10. Reject anything generic

## Gold principle
Great copy does not persuade. It reflects.
