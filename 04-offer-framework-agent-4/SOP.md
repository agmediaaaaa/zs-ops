# OFFER FRAMEWORK (AGENT 4 - CONVERSION LAYER)

## Objective
Turn the service into an offer the ICP wants now.

## Inputs required
- Strategy output
- Research output
- Client context

## Steps
1. Define core service and outcome
2. Extract real value: remove, enable, accelerate
3. Map ICP priorities, ignored metrics, and triggers
4. Apply value equation and optimize all variables
5. Select offer style: Mitchell, Nick, or Chris
6. Build core offer plus supporting variation
7. Add relevance, reduce friction, optional risk reversal
8. Reframe category and position the offer
9. Produce direct, situational, and pain-led variants
10. Validate fit, differentiation, and low-friction reply path

## Rules
- Offer is not a feature list
- ICP over product
- Context creates value
- Specificity wins
- Avoid vendor language
